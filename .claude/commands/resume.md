AllerReAlert 프로젝트 작업을 이어간다.

먼저 다음을 읽고 현재 상태 파악:
1. /docs/Create_prd.md, /docs/Generate_task.md, /README.md
2. git log --oneline -30 (최근 커밋 흐름)
3. 현재 브랜치명과 working tree 상태

읽은 후 다음 4가지를 보고:
1. 현재까지 완료된 마일스톤 (T-XXX 커밋 기준으로 추정)
2. 가장 최근에 작업 중이던 Task ID
3. 미완료된 작업이 있다면 무엇인지
4. 다음에 시작할 마일스톤·Task 추천

내가 "M_ 시작" 또는 "T-XXX부터 이어서"라고 답할 때까지 기다려라.
